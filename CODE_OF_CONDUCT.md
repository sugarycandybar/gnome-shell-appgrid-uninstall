# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive
- Assume good intent
- Give actionable technical feedback
- Focus on code and behavior, not individuals

## Unacceptable Behavior

- Harassment, hate speech, or discrimination
- Personal attacks or intimidation
- Publishing others' private information
- Disruptive or bad-faith conduct

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this Code of Conduct.

## Scope

This applies to project spaces (issues, PRs, discussions, reviews) and public representation of the project.
